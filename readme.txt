svn_autoupdate.bat
Батник для автоматического обновления папок из svn. Можно добавить в планировщик задач
Перед использованием открыть батник блокнотом и в 5-й строке после знака "равно" перечислить нужные пути через пробел.  Например, так:
set list="E:\SVN\Папка 1" "E:\SVN\Папка 1" "E:\SVN\Папка 1"

-------------------------

svn_autoupdate.pyw
Демон для автоматического обновления папок из svn. Перечислить папки в переменной rep_paths и закинуть файл в автозагрузку.
-------------------------

GIT_file_history.py

Достаточно дарг-эн-дропом перенести файл на иконку программы
