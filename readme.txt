Достаточно дарг-эн-дропом перенести файл на иконку программы

svn_autoupdate.pyw
Демон для автоматического обновления папок из svn. Перечислить папки в переменной rep_paths и закинуть файл в автозагрузку.

--------------------------

svn_client.pyw
Клиент SVN с графическим интерфейсом. Отличия от других клиентов:
- фильтры можно добавить в комбобокс и сохранить, чтобы не вводить их после каждого запуска;
- пути к папкам также хранятся в комбобоксе;
- диапазоны дат и ревизий не сбрасываются после нажатия на Refresh;
- можно получать сообщения при появлении новых ревизий на ветке;
- после обновления лога фокус смещается к текущей ревизии;
- текущая ревизия отображается в заголовке окна;
- прочие мелочи.
Можно собрать pyinstaller'ом или запускать через интерпретатор.
